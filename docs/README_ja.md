## 紹介

HMCLはオープンソースでクロスプラットフォーム対応のMinecraftランチャーです。Mod管理、ゲームカスタマイズ、Modローダーのインストール（Forge、NeoForge、Cleanroom、Fabric、Legacy Fabric、Quilt、LiteLoader、OptiFine）、Modパック作成、UIカスタマイズなど、さまざまな機能をサポートしています。

HMCLは優れたクロスプラットフォーム性能を持っています。Windows、Linux、macOS、FreeBSDなどの異なるオペレーティングシステムだけでなく、x86、ARM、RISC-V、MIPS、LoongArchなどのさまざまなCPUアーキテクチャにも対応しています。HMCLを使えば、さまざまなプラットフォームでMinecraftを簡単に楽しむことができます。

HMCLが対応しているシステムやCPUアーキテクチャについては、[この表](PLATFORM.md)をご参照ください。

## 貢献

HMCL はコミュニティ主導のオープンソースプロジェクトであり、コードの貢献や提案を歓迎しています。

HMCL の開発には、次の方法で参加できます。

- GitHub で [Issue を作成](https://github.com/HMCL-dev/HMCL/issues/new/choose)して、バグ報告や機能要望を行う。
- GitHub でリポジトリを fork し、[Pull Request を送信](https://github.com/HMCL-dev/HMCL/compare)してコードを貢献する。

貢献する前に、以下の内容を含む[貢献ガイド](./Contributing.md)をお読みください。

- [ソースコードから HMCL をビルドして実行する方法](./Contributing.md#build-hmcl)
- [デバッグオプションで HMCL の動作を調整する方法](./Contributing.md#debug-options)
